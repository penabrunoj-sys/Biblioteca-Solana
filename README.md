Se desarrolló un programa en la blockchain de Solana utilizando Rust y el framework Anchor Framework, en el cual se implementa una biblioteca virtual con sistema de venta de libros llamada Biblioteca. El proyecto permite crear una biblioteca asociada a una wallet, registrar libros con información como nombre, número de páginas y precio, visualizar el catálogo disponible, eliminar libros, cambiar su estado de disponibilidad y permitir que los usuarios compren libros dentro del sistema.

Durante el desarrollo se aplicaron conceptos fundamentales del ecosistema de Solana como la creación de cuentas en la blockchain, Program Derived Addresses (PDA), almacenamiento de estructuras de datos on-chain, validaciones de seguridad para controlar quién puede modificar la biblioteca y ejecución de instrucciones mediante transacciones. Además, el proyecto fue probado y ejecutado dentro de Solana Playground, lo que permitió interactuar con el contrato inteligente y verificar su funcionamiento en la red de desarrollo.

Este proyecto fue realizado como parte del proceso de aprendizaje y práctica en el desarrollo de smart contracts en Solana, adaptando el sistema original a una biblioteca virtual con gestión y venta de libros, con el objetivo de fortalecer habilidades en programación blockchain y cumplir con los requisitos necesarios para obtener la certificación de desarrollo en Solana.
